# 💬 Real-Time Chat Application

A real-time chat app using Node.js and Socket.io. Simple and elegant interface to connect and chat live with others.

## 📸 Screenshot
![Chat Screenshot](screenshots/chat.png)

## ✨ Features
- 🔴 Live messaging via WebSockets
- 🧑‍🤝‍🧑 User join/leave announcements
- 📱 Responsive chat UI
- 🖌️ Clean and modern layout

## 🛠 Tech Stack
- Node.js
- Express.js
- Socket.io
- HTML + CSS + JavaScript

## 📂 Project Structure
```
📁 chat-app/
├── public/
│   ├── index.html
│   ├── style.css
│   └── script.js
├── server.js
└── package.json
```

## ▶️ How to Run
1. Install Node.js
2. Clone the project:
   ```bash
   git clone https://github.com/yourusername/chat-app.git
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the server:
   ```bash
   node server.js
   ```
5. Visit `http://localhost:3000`
